# book_downloader

本着学习交流的心态写的，请勿做于商用。

新增：
  1. 添加代理 IP 池，可以通过代理 IP 池访问下载。
  2. 新增并发下载，增加下载效率。
  3. Cookies 自动获取，无需手动添加。
  4. 优化执行逻辑。
  5. 优化当访问失败后，自动尝试 3 次。
  6. 优化当模糊查询结果超过 16 个之后不显示的问题。
